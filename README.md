# 「zty-01」

## 项目简介

「zty-01」是一个专为学生设计的、无干扰的学习辅助设备。旨在提供一个纯粹、高效且符合学校规范的信息获取工具，帮助学生在禁用手机的场景下，依然能够便捷地管理课表和进行时间规划，从而更专注于学习本身。

## 硬件清单

- `ESP32-S3-DevKitC-1` 开发板
- 协议为 `SSD1306` 或 `SSD1305`、分辨率为 `128x64`、接口为 `I2C`、带有 4 个按键的 OLED 显示屏
- 面包版以及跳线

## 引脚

- 光照传感器 SCL: Pin 19
- 光照传感器 SDA: Pin 18
- 板载 LED: Pin 48
