# 「zty-01」

## 项目简介

「zty-01」是一个专为学生设计的、无干扰的墨水屏学习辅助设备。「zty-01」旨在提供一个纯粹、高效且符合学校规范的信息获取工具，帮助学生在禁用手机的场景下，依然能够便捷地管理课表和进行时间规划，从而更专注于学习本身。

## 核心特性

- 🛡️ 无干扰学习环境
  - 天生无蓝光、无背光，显示如同纸张，有效保护视力。
  - 网络功能仅用于同步时间、无游戏、无社交媒体，从根源上杜绝一切分心因素。
- 🧩 革命性模块化设计
  - 独创的墨水屏可拆卸设计：用户在连接供电刷新一次课表后，可以将墨水屏部分（含排线）单独从主控板上拆下。
  - 极致的便携性：拆下的墨水屏模块轻薄如纸，可轻松放入笔袋、口袋或书本之间，真正实现“一纸在手”，随时随地查看信息，完全无感无负担。
  - 纯粹的信息终端：拆下后墨水屏不具备任何电子功能，是真正的纯显示器，更符合校园环境的特殊要求。
- 📅 课表与信息展示
  - 清晰显示每日/每周课表，支持个性化排版。
  - 通过主控板连接电脑或手机刷新内容，方便快捷。
- ⏱️ 智能计时器
  - 考虑到三色墨水屏全屏刷新耗时1.8秒的特性，计时器优化为**每10秒刷新一次**。
  - 兼顾功能性与墨水屏的显示特性，最大程度降低刷新带来的视觉干扰。
- 🔋 电池供电
  - 回归最原始的设计，采用3节5号电池供电，无需充电器，可随时更换电池。
  - 结合ESP32-S3的深度睡眠模式，设备整体续航能力强劲。
- ⚙️ 核心主控：ESP32-S3
  - 采用强大的ESP32-S3作为主控芯片，性能稳定，支持Wi-Fi和蓝牙。
  - 为设备提供足够的处理能力和存储空间，并为未来的功能扩展打下基础。
- 🌐 开源硬件
  - 提供完整的代码和相关文档，鼓励创客、学生和开发者进行二次开发、学习和分享。
